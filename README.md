# VacationTracker

## Slack Command:

- /time_off: 

	+ from YYYY-MM-DD to YYYY-MM-DD, reason: text

	+ help

Example:

	+ /time_off from 2019-11-20 to 2019-11-21, reason: I got sick

	+ /time_off help

## Dashboard

- Users login/out by Google Oauth
- User role: employee, owner
- View remaining annual leave
- View leave history
